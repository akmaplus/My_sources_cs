  Класс Component является базовым классом для всех компонентов в среде CLR,
маршалинг которых выполняется по ссылке. Класс Component поддерживает удаленный
доступ и является унаследованным от класса MarshalByRefObject. Класс Component
предоставляет реализацию интерфейса IComponent. Класс MarshalByValueComponent
предоставляет реализацию интерфейса IComponent, маршалинг которого выполняются
по значению.


  Экземпляр класса Component можно поместить в любой объект, реализующий
интерфейсIContainer, после чего к этому контейнеру можно обращаться с запросами
и пользоваться его службами. Контейнер создает интерфейс ISite для каждого
содержащегося в нем объекта Component. Этот узел используется контейнером для
управления объектом Component, а сам контейнер используется объектом Component
для взаимодействия со своим контейнером.


  Рекомендуется, чтобы объект Component освобождал ресурсы явно, путем вызова
своего метода Dispose, не дожидаясь, пока это сделает автоматическая система
управления памятью с помощью неявного вызова метода Finalize. Когда
освобождается память, занимаемая объектом Container, все компоненты,
содержащиеся в этом объекте Container, также удаляются.
